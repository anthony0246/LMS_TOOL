Chatbot Usage Examples

Here are some practical examples of how to interact with the chatbot.

Example 1: Sending a Simple Message

To send a message, type the following:

const message = "Hi there, chatbot!";

Example 2: Sending a Message with Attachments

You can also send attachments using the file selector:

const attachment = document.querySelector('#file-input').files[0];

Example 3: Handling Responses

After sending a message, handle the response like this:

fetch('https://api.example.com/chatbot/send', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({ message: "Hello" }),
})
.then(response => response.json())
.then(data => console.log(data));
