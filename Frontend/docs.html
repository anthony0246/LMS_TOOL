Chatbot API Documentation

Welcome to the documentation. Here, you will find all the necessary information to integrate and use our chatbot effectively.

API Reference

To send a message using the chatbot API, use the following request:

POST https://api.example.com/chatbot/send
Headers:
  Content-Type: application/json
Body:
  {
    "message": "Hello, chatbot!"
  }

Response

The API will respond with a JSON object containing the chatbot's reply.
